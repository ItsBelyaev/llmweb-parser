------------------------------------------------------------------
ЗАПУСК ЛОКАЛЬНО
------------------------------------------------------------------

  git clone https://github.com/ItsBelyaev/llmweb-parser.git
  cd llmweb-parser

  python3.12 -m venv venv
  source venv/bin/activate          # macOS/Linux
  # venv\Scripts\activate           # Windows

  pip install -r requirements.txt
  pip install --upgrade "huggingface-hub>=1.0"
  playwright install chromium

  cp .env.example .env
  # вписать HUGGINGFACE_API_KEY (бесплатно: huggingface.co/settings/tokens)
  # опционально вписать OPENAI_API_KEY для GPT-4o-mini

  uvicorn app.main:app

Открыть в браузере: http://localhost:8000
Swagger UI:        http://localhost:8000/docs

------------------------------------------------------------------
ФУНКЦИОНАЛ
------------------------------------------------------------------

1. Прямое извлечение (POST /api/parse):
   LLM анализирует HTML и возвращает структурированные данные о товаре.

2. CSS-селекторы Parsmart (POST /api/selectors/generate, apply):
   Один вызов LLM → CSS-селекторы → сохранение в TOON-формате.
   Применение селекторов к новым URL без обращения к LLM.

3. Интерактивные действия (POST /api/interact):
   Универсальный поиск кнопок (эвристика + LLM-fallback) и
   автоматические клики через Playwright. Поддерживаемые действия:
   add_to_cart, buy_now, open_product, click_text, custom_selector.
